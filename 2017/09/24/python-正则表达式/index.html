python 正则表达式相关

匹配规则

  • d{3}表示匹配3个数字，例如'010'
  • \d{3,8}表示3-8个数字，例如'1234567'
  • *表示任意个字符（包括0个），
  • +表示至少一个字符
  • ?表示0个或1个字符
  • {n}表示n个字符
  • {n,m}表示n-m个字符
  • ^表示行的开头，^\d表示必须以数字开头
  • $表示行的结束，\d$表示必须以数字结束

匹配

  • 用r参数可以不考虑转义问题
  • match()方法判断是否匹配，如果匹配成功，返回一个Match对象，否则返回None

分组

  • m.group(index)
  • 0永远为原字符串
