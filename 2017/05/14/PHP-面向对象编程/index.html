PHP 面向对象相关

final关键字

final关键字只能用来标记类和方法

1、final标记的类不能被继承
2、final标记的方法不能被子类覆盖，是最终版本

static 和 const

1、static
（1）static的成员在类第一次被加载的时候分配空间，被实例化的对象共用，属于类，无法用$this访问
（2）在类内部访问static成员

（3）在类外部访问static成员

2、const
（1）const为在类内部定义常量的方法
（2）在类内部访问const属性

（3）在类外部访问const属性

trait

解决代码重用问题

public、protected、private

protected方法不能直接在继承的类的对象中访问，要在继承类中重新定义一个方法里面调用 $this->method
1、public声明的成员可以在任何位置调用
2、protected声明的成员只有在类的内部还有子类中可以调用
3、private声明的成员只有在类的内部可以调用
4、调用方法为

注：子类的权限只能大于等于父类

接口（Interface）

定义：类里面所有的方法都为抽象方法的类称为接口

1、接口内不能声明变量，但可以声明常量constant
2、所有成员都是public权限
3、声明接口使用关键字interface

4、定义接口的子类实现接口中全部抽象方法使用关键字implements

5、子类必须实现所有的抽象方法才能被实例化
6、一个类可以实现多个接口

7、在继承一个类的同时实现多个接口，先继承再实现

面向接口编程

在需要用到类作为参数的方法中将接口作为参数，过后传参数只需要传一个实现的接口的对象就可以

魔术方法

__get($name)

调用不存在的属性时自动调用，$name为调用的属性名

__invoke($name)

  • 以调用函数的方式调用一个对象时，自动调用
  • 匿名函数等于重写了闭包对象的__invoke()函数
  • array_map(闭包函数，array) 对array每个元素进行闭包函数操作

SplObjectStorage类

  • 实现了以对象为键的映射，可以直接删除指定对象，不用遍历整个集合

static::和self::

  • static::为静态绑定，不会解析为定义当前方法所在的类，而是实际运行的类
  • self::写在哪就调用哪个类
